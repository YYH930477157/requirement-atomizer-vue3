# UFER

## Definition

Row-level Register object holding the Brazil ABNT `UFER` (Value absolute) value per tariff, at logical name `1-0:132.8.x.255`. ABNT Appendix 9 (NBR 16968:2022) registers this object under two equivalent codes: the main code `1-0:132.8.x.255` (this entry) and the country-specific code `1-0:94.55.x.255` (separate entry).
